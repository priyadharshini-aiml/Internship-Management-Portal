# Internship Management Portal for College

## Overview
The Internship Management Portal is a Java-based web application designed to simplify and automate internship management in colleges.

## Objectives
- Provide a centralized platform for internship management
- Enable students to apply for internships online
- Allow companies to post internship opportunities
- Help administrators monitor and verify internships

## Modules
- Student Module
- Company Module
- Admin Module

## Technologies Used
- Java
- Spring Boot
- MySQL
- HTML, CSS

## Project Status
This project is currently under development as part of an academic curriculum.
